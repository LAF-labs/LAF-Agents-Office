---
description: Promote reviewed Notebook drafts into the shared Wiki
---

Inputs: agent slug, Notebook entry, target Wiki page.

Workflow:

1. Read the Notebook entry.
2. Search Wiki for related or contradictory pages.
3. Verify source, owner, date, and checks.
4. Resolve contradictions.
5. Use `notebook_promote` or the web promotion UI.
6. Add sources and keep the Notebook provenance.

Never auto-promote subconscious drafts.
